## Quick Start

Install alpacasay:

```bash
pip install alpacasay
```

Display a simple message:

```bash
alpacasay "Hello World!"
```

Use different alpaca types and colors:

```bash
alpacasay --alpaca happy --color green "I'm happy!"
alpacasay --alpaca thinking --color blue "Let me think..."
```
